# Tennis Kata (TypeScript)

A TypeScript implementation of the classic Tennis scoring kata.

## The Kata

Implement the scoring system for a tennis game:

- Points progress: Love → 15 → 30 → 40
- At 40–40, the score is "Deuce"
- From deuce, winning a point gives "Advantage"
- A player wins by reaching a scoring position with a two-point lead

## Getting Started

```bash
pnpm install
pnpm test
```
